# The Three LionGateOS Bible Editions

## Overview

The LionGateOS Bible project maintains three distinct editions.

Each edition has a different purpose and level of customization.

---

# 1. Enhanced Standard King James Version

## Purpose

A study-focused King James Version experience.

Includes:

- KJV Bible text
- Strong's Concordance support
- Bible study data
- Words of Jesus highlighting
- Additional study enhancements

This edition preserves the traditional KJV text while adding study resources.

---

# 2. LionGateOS Custom King James Version

## Purpose

A documented custom Bible edition developed by LionGateOS.

Includes:

- LionGateOS editorial revisions
- Documented wording changes
- Strong's Concordance support
- Words of Jesus highlighting
- Validation records
- Edition history

The Custom KJV is a protected custom edition and is not released as an open modification project.

See:

- `LICENSE_CUSTOM_KJV.md`

---

# 3. LionGateOS Custom KJV e-Sword Edition

## Purpose

A separately packaged edition designed for e-Sword-compatible use.

Includes:

- LionGateOS Custom KJV text
- e-Sword-compatible packaging
- Installation documentation

The e-Sword edition is distributed separately from the authoritative source files.

Installation instructions are documented separately.

---

# Source and Documentation

The project maintains documentation for:

- source information
- revision history
- validation
- export formats
- edition differences
- usage policies

The goal is transparency while preserving the integrity of each edition.
