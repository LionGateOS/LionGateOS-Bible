# Edition Structure

`source/growdaily/assets/bible/en_kjv.json`

`source/growdaily/assets/red_letters/jesus_speech_map.json`

`source/growdaily/assets/strong/`

## LionGateOS Custom KJV

Authoritative app text:

`source/growdaily/assets/bible/kjv_modified.json`

`source/growdaily/assets/red_letters/jesus_speech_map_custom.json`

`source/growdaily/docs/bible/`

## e-Sword edition

Target directory:

`editions/esword/`

This edition must be generated from the LionGateOS Custom KJV source and must never become the canonical editing source.
