# LionGateOS Bible Editions

This repository preserves and documents three related Bible distributions.

## Editions

### 1. Enhanced King James Version

Standard King James Version text enhanced with:

- Purple words of Jesus
- Strong's Concordance integration
- No LionGateOS textual revisions

### 2. LionGateOS Custom King James Version

The LionGateOS revised KJV text with:

- Documented textual changes
- Purple words of Jesus
- Strong's Concordance integration
- Editorial and validation records

### 3. LionGateOS Custom KJV for e-Sword

A future e-Sword-compatible distribution generated from the LionGateOS Custom KJV.

The e-Sword package is a generated output. It is not the authoritative source text.

## Repository principles

- Canonical source files remain separate from generated distributions.
- Every textual change must be documented.
- Generated files must identify the source version used.
- Validation results and checksums must accompany releases.
- GrowDaily app code remains in the separate GrowDaily repository.
